# frost-loading

* [API](#API)
* [Examples](#Examples)

## API
| Attribute | Type | Value | Description |
| --------- | ---- | ----- | ----------- |
| `type` | `string` || **default** - ripple effect loading indicator |
| | | `ring` | ring effect loading indicator |

## Examples

### Default - Ripple
```handlebars
{{frost-loading}}
```

### Type - Ring
```handlebars
{{frost-loading 
  type='ring'
}}
```
